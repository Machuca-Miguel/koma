# Q: auth flow

## Answer

Auth flow has three entry paths: register (POST /auth/register), local login (POST /auth/login via LocalAuthGuard+local.strategy), and Google OAuth (GET /auth/google via GoogleAuthGuard+google.strategy+callback). All paths produce a JWT. JwtAuthGuard protects all other controllers (ai, comics, collections, user-comics, isbndb, users, collection-series) via jwt.strategy.ts. AuthController (auth.controller.ts:L25) exposes register L33, login L42, getProfile L51, googleLogin L58, googleCallback L65.
